## 📖 About

This project recreates a modern admin dashboard interface using CSS Grid and Flexbox. The goal was to practice advanced CSS layout techniques without using any frameworks.

---

## ✨ Features

- Responsive dashboard layout
- Sidebar navigation
- Header with search bar
- User profile section
- Project cards
- Announcements panel
- Trending users section
- SVG icons
- CSS Grid & Flexbox layout
- Clean and modern UI

---

## 🛠️ Built With

- HTML5
- CSS3
- CSS Grid
- Flexbox
- SVG Icons

## 📚 What I Learned

While building this project, I practiced:

- Nested Grid layouts
- Flexbox alignment
- Pseudo-elements (`::before`)
- Attribute selectors
- Responsive spacing
- CSS positioning
- Border styling
- SVG icon integration
- Layout planning
